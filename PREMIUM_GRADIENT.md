# Premium Dark Gradient — Обновление цветовой схемы баннера

**Дата**: 14 апреля 2026

---

## 🎨 Что изменилось

### До:
```scss
background-image: linear-gradient(
  45deg, 
  var(--creamy) 0%,    // Светло-бежевый
  #051858 24%,         // Темно-синий
  #235ac2 49%,         // Яркий синий
  #ffffff 75%,         // Белый
  var(--creamy) 100%   // Светло-бежевый
);
filter: drop-shadow(0px 0px 7px #d8bb91); // Золотистое свечение
```

**Проблема**: Слишком яркий, голубой градиент не сочетается с темной темой сайта.

---

### После:
```scss
background-image: linear-gradient(
  135deg,
  #0a0e1a 0%,    // Темно-синий (основной цвет сайта)
  #1a2332 25%,   // Чуть светлее темно-синий
  #2d3748 50%,   // Slate gray
  #4a5568 75%,   // Lighter slate
  #0a0e1a 100%   // Возврат к темно-синему
);
filter: 
  drop-shadow(0px 4px 20px rgba(77, 212, 208, 0.15))  // Легкий cyan glow
  drop-shadow(0px 0px 40px rgba(10, 14, 26, 0.6));    // Глубокая тень
```

**Результат**: Премиальный темный градиент, который органично вписывается в дизайн.

## 🌈 Цветовая палитра

### Основные цвета:
- `#0a0e1a` — Deep Navy (основной цвет сайта)
- `#1a2332` — Midnight Blue
- `#2d3748` — Slate Gray
- `#4a5568` — Cool Gray

### Акценты:
- `rgba(77, 212, 208, 0.15)` — Subtle Cyan Glow (из основной палитры сайта)
- `rgba(10, 14, 26, 0.6)` — Deep Shadow

---

## 💎 Премиальные улучшения

### 1. Монохромная база
Вместо ярких цветов используется градация одного оттенка (темно-синий → серый → темно-синий).

### 2. Tinted shadows
Тени окрашены в цвета палитры:
- **Cyan glow**: Легкое свечение акцентным цветом сайта
- **Deep shadow**: Глубокая тень в тон фона

### 3. Diagonal gradient (135deg)
Диагональный градиент создает больше динамики, чем стандартный 45deg.

### 4. Smooth transitions
Плавные переходы между оттенками без резких скачков.
